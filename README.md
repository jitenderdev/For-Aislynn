# For-Aislynn

## How to run

1. To run the project
  
  ```
  npm install
  ```
  
  If running on Mac machine use below command to install pods for iOS
  ```
  npx pod-install
  ```
  
2. To build Android 

  ```
  npx react-native run-android
  ```
  
3. To build iOS (Works on only Mac machine)

  ```
  npx react-native run-ios
  ```
